  • Make sure git is installed in your computer and you have a GitHub account
  • Navigate to the directory in your computer where you want to create your project and create a folder
  • Create necessarily files for your project
  • In the terminal navigate to the directory of the project
  • In terminal type 'git init' to initialize a git repository
  • Type 'git add .' to stage your files
  • Type 'git commit -m "your descriptive commit message" ' to commit changes you have made
  • Go to GitHub and create a new repository
  • Copy the SSH link for the GitHub repo
  • Go to terminal and type 'git remote add origin "YOUR SSH LINK" '
  • Type 'git push origin master' to push your local changes to your remote repo on GitHub
